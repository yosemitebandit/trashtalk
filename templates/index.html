the trashtalk api

Does your robot talk? Does it need a bit more personality? Try the trashtalk api: talking smack as a service.

Access any of these endpoints for a random message you can integrate into your code:

  • /victory
  • /defeat
  • /greeting
  • /init
  • /ready
  • /idle
  • /laughter
  • /sobbing

Have a mac? Try it out with one line:

curl -s http://trashtalk.oakmachine.com/victory | say
